# 📊 Student Score System - C++

A simple student score management system built with **C++** using **File I/O (fstream)** to save and read data.

---

## ✨ Features

- ✅ Input student name and scores
- ✅ Display all input scores
- ✅ Calculate total, average, highest, and lowest score
- ✅ Save results to `student_scores.txt` file
- ✅ Read and display file content automatically

---

## 🛠️ Technologies Used

- C++
- Standard Template Library (STL): `vector`, `iostream`, `fstream`
